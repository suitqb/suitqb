Final-year CS engineering student, mostly interested in AI and what it can do when applied to real systems — perception, robotics, embodied agents.

**Currently:** Researching VLM-based semantic scene understanding for autonomous driving — building the perception layer that feeds a pedestrian intention prediction pipeline (Research Intern @ ALTEN Innovation Lab).

---

### Projects I'm proud of

**[`Euros`](https://github.com/suitqb/Euros)** — Built a Large Language Model from scratch using PyTorch. No wrapper, no pretrained backbone — the full thing, architecture included.

**[`VLM-LLM-Robotics`](https://github.com/suitqb/VLM-LLM-Robotics)** — Survey of how LLMs and VLMs are used in robotics, plus a real implementation: deployed VLMaps on a TurtleBot with the goal of making it work in the real world.

**[`Vision_Project`](https://github.com/suitqb/Vision_Project)** — Diffusion model (DDPM) implemented from scratch: UNet, EMA, full training loop.

**[`FederatedLearning`](https://github.com/suitqb/FederatedLearning)** — Federated learning experiments — distributed training without centralizing data.

**Interested in:** LLM / VLM applications · Spatio-temporal graph modeling · AI in robotics & embodied agents
